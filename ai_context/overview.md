# Project Overview

This is an Android application using MVVM architecture.

Modules:

- app_customer: user-facing app
- app_admin: admin panel

Tech stack:

- Kotlin
- Retrofit (API)
- Room (local database)
- Hilt (DI)

Main layers:

- UI: Activity / Fragment
- ViewModel: handles UI logic
- Repository: business logic + data coordination
- Data: API + Database
